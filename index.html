On Time Attendance

The OnTimeAttendance is an Android application built to manage student attendance for tuition classes.
This application has a very user friendly interface which is built using Java and Android Studio, with Firebase used as the backend (which will later be replaced with an SQL database).
Students can register and login to this app and can mark their attendances by scanning a QR code provided by the teacher. This App has Multi-user access with different permissions for Teachers so that they can check records for previous classes.

Diary Management System

The Personal Diary Management System is a simple C language program that allows users to add, search for, and display diary records. Users can enter the date, time, place, duration of stay, and a note for each record, which is stored in a linked list.
The program features a tabular display of all saved records and the option to search for records by date. Future improvements could include adding the ability to edit existing records, improving the user interface, and exporting data to a file. Overall, this project offers a straightforward and efficient solution for personal diary management.
